Read my BuJo logs and give me a focused coaching session.

Run this shell command to get my logs:
```
bujo coach --human
```

Note: entries are stored as plain ASCII (t, x, >, k, n, e, *) in markdown files.
The TUI renders these as Unicode symbols visually.

Then read the output and do the following:

1. If it says "Not enough data": tell me the vault is new. Suggest running `bujo template morning`.

2. If there's data:
   - Read momentum, streak, stuck tasks, kill patterns, priority alignment
   - Note most productive time if available
   - Give a direct coaching synthesis — not a summary, an interpretation
   - End with exactly ONE question. Not multiple. One.

Tone: direct, honest, no fluff. Short paragraphs. No bullet points.
